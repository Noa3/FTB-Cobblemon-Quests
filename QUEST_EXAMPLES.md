# Quest Examples - FTB Cobblemon v3.0

This document showcases examples of the new quests added in v3.0.

## Catching Quests (Examples)

### Specific Pokémon
- **Pika Pika!** - Catch Pikachu, one of the most popular Pokémon
- **Evolution Master** - Catch Eevee, the Evolution Pokémon  
- **Flame Wing** - Catch Charizard, the powerful Fire/Flying type
- **Genetic Power** - Catch Mewtwo, the legendary Psychic type
- **Atrocious Power** - Catch Gyarados, evolved from Magikarp
- **Dragon Master** - Catch Dragonite, the dragon Pokémon
- **Aura Guardian** - Catch Lucario, the Fighting/Steel type
- **Land Shark** - Catch Garchomp, the Dragon/Ground type

### Level Tiers
- **Intermediate Catcher** - Catch a level 25+ Pokémon
- **Advanced Catcher** - Catch a level 40+ Pokémon
- **Expert Catcher** - Catch a level 50+ Pokémon
- **Elite Catcher** - Catch a level 60+ Pokémon
- **Master Catcher** - Catch a level 75+ Pokémon
- **Ultimate Catcher** - Catch a level 90+ Pokémon
- **Perfect Catch** - Catch a level 100 Pokémon

### Multiple Catches
- **Fish Farm** - Catch 5 Magikarp
- **Bug Collector** - Catch 10 Caterpie
- **Bird Watching** - Catch 5 Pidgey
- **Cave Patrol** - Catch 10 Zubat
- **Evolution Collection** - Catch 3 Eevee
